## 🚀 使用方法

### 方法1：直接部署到服务器

1. **将整个 `deploy_web` 目录上传到服务器**
2. **SSH连接到服务器**
3. **进入目录并安装依赖**
   ```bash
   cd deploy_web
   pip install -r requirements_web.txt
   ```
4. **启动应用**
   ```bash
   streamlit run ali_online.py --server.port 8501 --server.address 0.0.0.0
   ```
5. **访问应用**: http://服务器IP:8501

### 方法2：使用Docker部署

```bash
# 构建镜像
docker build -t ali-business-web .

# 运行容器
docker run -d -p 8501:8501 --name ali-app ali-business-web
```

### 方法3：使用Streamlit Cloud（最简单）

1. 将代码推送到GitHub
2. 访问 https://streamlit.io/cloud
3. 连接GitHub仓库
4. 选择 `ali_online.py` 作为主文件
5. 自动部署！

## 🔧 常见问题

### Q: 如何修改端口？
A: 编辑 `.streamlit/config.toml` 文件中的 `port` 值

### Q: 如何后台运行？
A: 使用 `nohup` 或 `screen`:
```bash
nohup streamlit run ali_online.py --server.port 8501 --server.address 0.0.0.0 > app.log 2>&1 &
```

### Q: 如何配置Nginx反向代理？
A: 参考 `README.md` 中的Nginx配置示例
